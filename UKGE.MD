# UKGE Running

# Prerequisite
- Install Anaconda
- Install Visual Studio Code

## Step: Import custom Anaconda Environment
This environment has panda, scikit-learn, tensorflow preinstalled

## Step: 

There are two models `logi` and `rect` which we can choose from. You can run the model with the following command:

```
python run/run.py -m rect --data cn15k --lr 0.01 --batch_size 300
```
